# Arbeitszeit

Eine kleine private Stempeluhr fuer GitHub Pages und iPhone Safari.

## Nutzung

1. `index.html`, `manifest.json`, `sw.js` und `icon.svg` in ein GitHub-Repository hochladen.
2. In GitHub unter `Settings > Pages` die Veroeffentlichung aus dem `main` Branch aktivieren.
3. Die GitHub-Pages-Adresse auf dem iPhone in Safari oeffnen.
4. In Safari `Teilen > Zum Home-Bildschirm` waehlen.

Die Daten werden nur lokal im Browser des iPhones gespeichert. Sie werden nicht an GitHub oder einen Server gesendet.

## Funktionen

- Kommen und Gehen per grossem Button stempeln
- Tagesnotiz speichern
- Tagesdauer und Monatsstatistik anzeigen
- CSV exportieren
- JSON-Backup exportieren und wieder importieren
- Letzten Eintrag oder alle Daten loeschen

## Wichtig

Vor einem iPhone-Wechsel oder vor dem Loeschen von Safari-Daten immer zuerst ein JSON-Backup exportieren. Sonst sind die lokal gespeicherten Zeiten weg.
